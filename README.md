## What is salesforce platform?
The Salesforce Platform is a cloud-based application development platform used to build and manage business applications.


## Explain: App-
An App in Salesforce is a collection of related tabs, objects, and features designed for a specific business purpose.

## Object-
An Object is like a database table used to store information.

## Tab-
A Tab is a user interface element used to open and access objects or pages in Salesforce.

## Difference:Configuration vs coding?
## Configuration
Done using clicks and settings
Does not require programming knowledge
Easier and faster to implement
Uses built-in Salesforce tools
Easy to maintain and update
Limited to standard functionalities
## Coding
Done using programming languages
Requires coding knowledge
Used for complex customizations
Provides more flexibility and control
Development takes more time
Requires testing and debugging
